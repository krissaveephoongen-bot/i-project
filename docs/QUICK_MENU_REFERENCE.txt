🚀 GETTING STARTED
─────────────────────────────────────────────────────────────────────────────

1. Login to application
   → Navigate to /auth/login

2. Access Dashboard
   → Click "Dashboard" in sidebar
   → URL: /dashboard

3. View Projects
   → Click "Projects" in sidebar
   → Choose from:
      • All Projects
      • Project Table
      • My Projects

4. Manage Resources
   → Click "Resources" in sidebar
   → View team members and availability

5. Track Costs
   → Click "Time & Expenses" → "Cost Management"
   → Monitor budget vs actual

6. View Analytics
   → Click "Analytics" → "Analytics Dashboard"
   → Review KPIs and metrics
