# ระบบติดตาม Manday และค่าใช้จ่ายโครงการ

## Manday Calculation

### สูตรคำนวณ Manday
```
Manday = ชั่วโมงทำงาน / 8
```

### ตัวอย่าง
- ทำงาน 8 ชั่วโมง = 1.00 manday
- ทำงาน 4 ชั่วโมง = 0.50 manday
- ทำงาน 2 ชั่วโมง = 0.25 manday

## โครงสร้างข้อมูล User (สำหรับคำนวณต้นทุน)

### ข้อมูลที่ต้องมี
- Username: ชื่อผู้ใช้
- Email: อีเมล
- Role: บทบาท (admin, manager, employee)
- Department: แผนก
- HourlyRate: อัตราค่าแรงต่อชั่วโมง (บาท)
- Status: สถานะ (active, inactive)

### การคำนวณต้นทุน Manday
```
ต้นทุน Manday = HourlyRate × 8 ชั่วโมง
ต้นทุนงาน = Manday × ต้นทุน Manday
```

## ค่าใช้จ่ายโครงการ

### ประเภทค่าใช้จ่าย
1. การเดินทาง
2. อาหาร
3. ที่พัก
4. อุปกรณ์
5. ซอฟต์แวร์
6. ค่าบริการ
7. อื่นๆ

### ข้อมูลที่บันทึก
- Date: วันที่เกิดค่าใช้จ่าย
- ProjectId: โครงการที่เกี่ยวข้อง
- Category: หมวดหมู่ค่าใช้จ่าย
- Amount: จำนวนเงิน (บาท)
- Description: รายละเอียด
- Status: สถานะการอนุมัติ (pending, approved, rejected)

## กระบวนการอนุมัติ

### Work Log
1. พนักงานบันทึกงานประจำวัน → Status: pending
2. ผู้จัดการ/Admin อนุมัติ → Status: approved
3. หากปฏิเสธ → Status: rejected

### Expense
1. พนักงานบันทึกค่าใช้จ่าย → Status: pending
2. ผู้จัดการ/Admin อนุมัติ → Status: approved
3. หากปฏิเสธ → Status: rejected

## การติดตามต้นทุนโครงการ

### ต้นทุนรวมของโครงการ
```
ต้นทุนรวม = ต้นทุน Manday ทั้งหมด + ค่าใช้จ่ายอื่นๆ ทั้งหมด
```

### Manday ที่ใช้ไปในโครงการ
```
Total Manday = Σ(Manday ของแต่ละ Work Log ที่ approved)
```

### งบประมาณคงเหลือ
```
งบประมาณคงเหลือ = งบประมาณโครงการ - ต้นทุนรวมที่ใช้ไป
```
