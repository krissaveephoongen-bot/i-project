## 📋 Quick Reference Sections

### Status Indicators (Always look here first)
- ✅ Green: Connected
- ⚠️ Orange: Disconnected
- ❌ Red: Error

### API Endpoints (For API calls)
- `/api/health/db` - Full check (3 retries)
- `/api/health/db/simple` - Quick test (2 retries)
- `/api/health/db/status` - Current status (instant)

### Component Files (For integration)
- `src/components/DatabaseStatus.tsx` - Status widget
- `src/pages/DatabaseStatusPage.tsx` - Status page
- `src/hooks/useDatabaseStatus.ts` - React hook
- `src/services/databaseService.ts` - API service

### Retry Times (For performance)
- Health Check: 1s + 2s + 4s = 7s max
- Simple Test: 1s + 2s = 3s max
- Manual: 1s + 2s + 4s + 8s + 16s = 31s max
