## 🔑 Key Concepts

### PIN Verification
- 6-digit numeric PIN
- Default: `123456` (change immediately!)
- Max 5 failed attempts
- Automatic lockout after failure
- See: ADMIN_CONSOLE_QUICK_FIX.md → PIN Verification

### Session Management
- 1-hour session duration
- sessionStorage-based
- Auto-expiry with warning
- Re-authentication required
- See: ADMIN_CONSOLE_SETUP.md → Session Management

### Route Protection
- Admin role required
- PIN verification required
- Auto-redirects for unauthorized
- Session validation on every access
- See: ADMIN_CONSOLE_SETUP.md → Route Protection

### API Integration
- Metrics endpoint
- Health status endpoint
- Logs endpoint
- Maintenance endpoints
- See: ADMIN_CONSOLE_SETUP.md → API Endpoints
