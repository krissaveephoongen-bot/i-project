## สำหรับพนักงานและผู้จัดการ (User Portal)

### 1. Dashboard (หน้าแรก)
- **ดู KPI ส่วนตัว**: Utilization, Pending Tasks, Project Health, Budget Burn
- **Clock In/Out**: บันทึกเวลาเข้า-ออกงาน
- **Quick Actions**: เข้าถึงฟังก์ชันหลักได้อย่างรวดเร็ว

### 2. Projects (โครงการของฉัน)
- **ดูโครงการที่เกี่ยวข้อง**: รายการโครงการที่คุณมีส่วนร่วม
- **คลิกดูรายละเอียด**: ดูความคืบหน้า งบประมาณ และงานต่างๆ
- **Project Overview**: คลิก "View Details" เพื่อดูกราฟ S-Curve และสถิติโครงการ

### 3. Timesheet (บันทึกเวลางาน)
- **เลือกวันที่**: เลือกวันที่ต้องการบันทึกงาน
- **เลือกประเภทงาน**: 
  - Project (งานโครงการ) - ต้องระบุโครงการและงาน
  - Office (งานออฟฟิศ)
  - Other (งานอื่นๆ)
- **กรอกชั่วโมง**: ระบุจำนวนชั่วโมงที่ทำงาน
- **บันทึก**: ระบบจะคำนวณ Manday อัตโนมัติ (ชั่วโมง ÷ 8)

### 4. Expenses (บันทึกค่าใช้จ่าย)
- **เพิ่มค่าใช้จ่าย**: คลิก "Add Expense"
- **เลือกโครงการ**: เลือกโครงการที่เกี่ยวข้อง
- **ระบุจำนวนเงิน**: กรอกจำนวนเงินเป็นบาท (฿)
- **เลือกหมวดหมู่**: เช่น travel, food, accommodation, equipment, software, service, other
- **รอการอนุมัติ**: ค่าใช้จ่ายจะแสดงสถานะ "Pending" จนกว่าจะได้รับการอนุมัติ

### 5. Task Board (กระดานงาน)
- **ดูงานทั้งหมด**: แสดงงานที่มอบหมายให้คุณในรูปแบบ Kanban
- **4 คอลัมน์**: To Do, In Progress, Review, Completed
- **กรองตามโครงการ**: เลือกดูงานของโครงการใดโครงการหนึ่ง
- **ดูรายละเอียดงาน**: คลิกที่การ์ดงานเพื่อดูข้อมูลเพิ่มเติม

### 6. Reports (รายงาน)
- **ดูสถิติส่วนตัว**: ชั่วโมงการทำงาน, ค่าใช้จ่าย, Utilization
- **ดูประสิทธิภาพโครงการ**: ความคืบหน้าโครงการที่เกี่ยวข้อง

## สำหรับผู้ดูแลระบบ (Admin Console)

### 1. Dashboard (หน้าแรกแอดมิน)
- **ภาพรวมทั้งระบบ**: สถิติโครงการ งบประมาณ ค่าใช้จ่าย
- **กราฟวิเคราะห์**: แสดงข้อมูลเชิงลึกของโครงการทั้งหมด

### 2. User Management (จัดการผู้ใช้)
- **เพิ่มผู้ใช้ใหม่**: คลิก "Add User"
- **กรอกข้อมูล**: ชื่อ, อีเมล, แผนก, ตำแหน่ง, บทบาท
- **กำหนด Hourly Rate**: สำหรับคำนวณต้นทุนแรงงาน
- **แก้ไข/ลบ**: จัดการข้อมูลผู้ใช้ที่มีอยู่

### 3. Customer Management (จัดการลูกค้า)
- **เพิ่มลูกค้า**: บันทึกข้อมูลลูกค้า/องค์กร
- **ประเภทลูกค้า**: Government, Private, Individual
- **ข้อมูลติดต่อ**: ชื่อผู้ติดต่อ, อีเมล, เบอร์โทร, ที่อยู่

### 4. Projects (จัดการโครงการ)
- **สร้างโครงการใหม่**: คลิก "New Project"
- **กรอกข้อมูลโครงการ**:
  - ข้อมูลพื้นฐาน: รหัส, ชื่อ, คำอธิบาย
  - กำหนดเวลา: วันที่เริ่ม-สิ้นสุด
  - งบประมาณ: ระบุเป็นบาท
  - ทีมงาน: ผู้จัดการ, สมาชิก
  - ลูกค้า: เลือกลูกค้า/หน่วยงาน
- **สร้าง Tasks**: แบ่งงานย่อยพร้อมกำหนด Weight (รวม 100%)
- **ติดตาม S-Curve**: วิเคราะห์ความคืบหน้าจริงเทียบแผน

### 5. Analytics (วิเคราะห์ข้อมูล)
- **Summary Cards**: โครงการทั้งหมด, โครงการที่กำลังดำเนินการ, งบประมาณรวม, ค่าใช้จ่ายรวม
- **S-Curve Analysis**: เปรียบเทียบความคืบหน้าตามแผนกับจริง
- **Project Overview Table**: ตารางสรุปโครงการทั้งหมด

### 6. Task Board (กระดานงานแอดมิน)
- **ดูงานทั้งหมด**: งานของทุกโครงการและทุกคน
- **กรองตามโครงการ**: เลือกดูงานของโครงการใดโครงการหนึ่ง
- **จัดการงาน**: แก้ไข สถานะ และความคืบหน้า

### 7. Approvals (อนุมัติงาน)
- **อนุมัติ Timesheet**: ตรวจสอบและอนุมัติบันทึกเวลางาน
- **อนุมัติค่าใช้จ่าย**: ตรวจสอบและอนุมัติค่าใช้จ่ายโครงการ
- **สถานะ**: Pending, Approved, Rejected

### 8. Reports (รายงานแอดมิน)
- **รายงานโครงการ**: สถิติและประสิทธิภาพโครงการ
- **รายงานทีม**: Workload ของ PM และทีมงาน
- **รายงานการเงิน**: งบประมาณ ค่าใช้จ่าย Invoice

## คำศัพท์และแนวคิดสำคัญ

### S-Curve
- **Planned Progress**: ความคืบหน้าตามแผน (คำนวณจาก Task Weight และวันที่กำหนด)
- **Actual Progress**: ความคืบหน้าจริง (คำนวณจากงานที่เสร็จแล้ว)
- **SPI (Schedule Performance Index)**: Actual ÷ Planned
  - SPI > 1.0 = ดำเนินการเร็วกว่าแผน
  - SPI = 1.0 = ตามแผน
  - SPI < 1.0 = ช้ากว่าแผน

### Task Weight
- น้ำหนักของงานในโครงการ (เป็น %)
- **กฎสำคัญ**: Task Weight ทั้งหมดในโครงการต้องรวมเป็น 100%
- ใช้คำนวณความคืบหน้าโครงการ

### Manday
- จำนวนวันทำงานของคน 1 คน
- **สูตร**: Manday = Hours ÷ 8
- ตัวอย่าง: ทำงาน 16 ชั่วโมง = 2 Manday

### Utilization
- เปอร์เซ็นต์การใช้เวลาทำงาน
- **เป้าหมาย**: 40 ชั่วโมง/สัปดาห์ = 100%
- **สูตร**: (ชั่วโมงที่บันทึก ÷ 40) × 100%

## เคล็ดลับการใช้งาน

### สำหรับพนักงาน
1. **บันทึก Timesheet ทุกวัน**: อย่ารอสรุปท้ายสัปดาห์
2. **ระบุโครงการและงานให้ชัดเจน**: ช่วยให้ติดตามความคืบหน้าได้แม่นยำ
3. **บันทึกค่าใช้จ่ายทันที**: พร้อมแนบหลักฐาน
4. **อัปเดตความคืบหน้างาน**: เปลี่ยนสถานะงานเมื่อมีความคืบหน้า

### สำหรับผู้จัดการโครงการ
1. **กำหนด Task Weight อย่างรอบคอบ**: ต้องสะท้อนความสำคัญของงานจริง
2. **ตรวจสอบ S-Curve เป็นประจำ**: เพื่อติดตามความคืบหน้า
3. **ติดตามงบประมาณ**: ควบคุมค่าใช้จ่ายไม่ให้เกินงบ
4. **อนุมัติ Timesheet และ Expenses อย่างสม่ำเสมอ**: อย่าให้สะสม

### สำหรับแอดมิน
1. **ตั้งค่าผู้ใช้และลูกค้าก่อน**: ก่อนเริ่มสร้างโครงการ
2. **กำหนด Hourly Rate**: เพื่อคำนวณต้นทุนแรงงาน
3. **ใช้ Analytics Dashboard**: เพื่อภาพรวมและตัดสินใจ
4. **สร้าง Tasks ให้ครบ**: และกำหนด Weight ให้รวม 100%

## การแก้ปัญหาเบื้องต้น

### ไม่สามารถเข้าสู่ระบบ
- ตรวจสอบอีเมลและรหัสผ่าน
- ลอง refresh หน้าเว็บ
- ติดต่อแอดมินเพื่อ reset รหัสผ่าน

### Task Weight ไม่เท่ากับ 100%
- ตรวจสอบ Weight ของแต่ละ Task
- ปรับ Weight ให้รวมเป็น 100% พอดี
- ระบบจะแจ้งเตือนถ้ารวมไม่ถึงหรือเกิน 100%

### ค่าใช้จ่ายเกินงบประมาณ
- ระบบจะเตือนเมื่อจะเกินงบ
- ตรวจสอบความจำเป็นของค่าใช้จ่าย
- ปรึกษาผู้จัดการโครงการก่อนบันทึก

### ไม่พบโครงการหรืองาน
- ตรวจสอบว่าได้รับมอบหมายหรือไม่
- ใช้ฟีเจอร์กรองหรือค้นหา
- ติดต่อแอดมินเพื่อขอสิทธิ์เข้าถึง
