                         HOW TO USE
================================================================================

STEP 1: Change Default PIN
------------------------------
Edit: src/config/admin-config.ts

Find:
  PIN: '123456',

Change to your custom 6-digit PIN:
  PIN: '987654',


STEP 2: Restart Server
------------------------------
npm start
# or
npm run dev


STEP 3: Access Admin Console
------------------------------
1. Go to sidebar menu
2. Look for "Admin Console" (red button with 🔐)
3. Click on it
4. Enter 6-digit PIN
5. Click "เข้าสู่ระบบ" or press Enter
6. Admin Console opens in new tab


STEP 4: Security (Optional)
------------------------------
Configure security settings in: src/config/admin-config.ts
- Enable logging
- Enable notifications
- Set admin email
- Configure rate limiting

================================================================================
                       ENVIRONMENT VARIABLES
================================================================================

For production security, use environment variables:

Create .env file:
  REACT_APP_ADMIN_PIN=987654

Update config file to use it:
  export const getAdminPIN = (): string => {
    return process.env.REACT_APP_ADMIN_PIN || adminConfig.PIN;
  };

Now PIN is not hardcoded in source code!
