═══════════════════════════════════════════════════════════════════════════════
  SUPPORT & TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Issue: "Routes not found"
Solution: Restart backend (npm run dev)

Issue: "Can't connect to database"
Solution: Check DATABASE_URL in .env file

Issue: "Tasks not loading"
Solution: Select a project first

Issue: "Can't approve timesheets"
Solution: Check if you have manager role

Issue: "API returns 404"
Solution: Check routes in server/app.js

For more help:
  1. Check SETUP_PROJECT_MANAGEMENT.md
  2. Check USER_GUIDE_PROJECT_MANAGEMENT.md
  3. Check API_REFERENCE.md
  4. Review component source code
  5. Check browser console for errors
