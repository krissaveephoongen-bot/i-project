# Timesheet Usage Guide - ไทม์ชีทคำแนะนำการใช้งาน

## ส่วนประกอบหลัก (Main Components)

### 1. ปุ่มควบคุมด้านบน (Top Controls)

```
[◄  เดือนปัจจุบัน  ►]  [สถานะ]  [ชั่วโมงรวม]
                          [แก้ไขเวลา]  [เพิ่มรายการใหม่]  [ส่งอนุมัติ]
```

#### ปุ่มแต่ละอันมีหน้าที่:

**ปุ่ม "แก้ไขเวลา" (Edit Mode)**
- สถานะ: ปุ่ม outline (ปกติ) → secondary (กำลังแก้ไข)
- หน้าที่: เปิด/ปิดโหมดแก้ไข
- ใช้เมื่อ: ต้องการแก้ไขเวลาของวันที่มีข้อมูลแล้ว
- ข้อกำหนด: สามารถใช้เมื่อสถานะเป็น Draft, Rejected, หรือ Submitted (ถ้าเป็น admin/manager)

**ปุ่ม "เพิ่มรายการใหม่" (Add Entry)**
- สีเขียว (Green)
- หน้าที่: เพิ่มรายการไทม์ชีทใหม่โดยไม่ต้องคลิกที่วัน
- ข้อมูลเริ่มต้น:
  - วันที่: วันนี้
  - โครงการ: โครงการแรกในรายการ
  - ชั่วโมง: 0
  - สำเร็จ ตัวอักษร: ว่างเปล่า
- ข้อกำหนด: สามารถใช้ได้ในสถานะเดียวกับ "Edit Mode"

**ปุ่ม "ส่งอนุมัติ" (Submit)**
- สีน้ำเงิน (Blue)
- หน้าที่: ส่งไทม์ชีทเดือนนี้สำหรับอนุมัติ
- ข้อมูลที่ส่ง:
  - วันเริ่มต้นของเดือน
  - วันสิ้นสุดของเดือน
  - รวมชั่วโมงทั้งหมด
- คำเตือน: เมื่อส่งแล้ว จะไม่สามารถแก้ไขได้จนกว่าหัวหน้างานพิจารณา
- ข้อกำหนด: แสดงเมื่อสถานะ = Draft เท่านั้น

## วิธีการใช้งาน (How to Use)

### ใช้งาน 1: เพิ่มรายการไทม์ชีทวิธีที่ 1 - ใช้ปุ่ม "เพิ่มรายการใหม่"

```
1. คลิก "เพิ่มรายการใหม่"
   ↓
2. Modal เปิดขึ้นมา (เริ่มต้นด้วยวันนี้)
   ↓
3. (ตัวเลือก) เปลี่ยนวันที่ ← ถ้าต้องการบันทึกเป็นวันอื่น
   ↓
4. (ตัวเลือก) เปลี่ยนโครงการ ← ถ้าต้องการเลือกโครงการอื่น
   ↓
5. กรอกชั่วโมง (บังคับ)
   ↓
6. (ตัวเลือก) เลือก Task ← ถ้าต้องการ
   ↓
7. (ตัวเลือก) เลือก Start/End Time ← ถ้าต้องการบันทึกเวลา
   ↓
8. (ตัวเลือก) เพิ่มคำอธิบาย
   ↓
9. คลิก "+ เพิ่มแถว" หรือ "บันทึก"
```

**ข้อดี:**
- ไม่ต้องคลิกที่วันในตาราง
- มีค่าเริ่มต้นสำหรับวันและโครงการ
- เหมาะสำหรับการเพิ่มรายการอย่างรวดเร็ว

---

### วิธีที่ 2: เพิ่มรายการโดยคลิกที่วันในตาราง

```
1. คลิก "แก้ไขเวลา" เพื่อเปิดโหมดแก้ไข
   ↓
2. ค่อนด้างแต่อเดือน modal แสดง "กำลังแก้ไข"
   ↓
3. คลิกที่วันที่ต้องการเพิ่มเวลา (ตารางกลับมา)
   ↓
4. Modal ของวันนั้นจะเปิดขึ้น
   ↓
5. ทำตามขั้นตอนเดียวกับ "วิธีที่ 1" (ขั้นตอน 5-9)
```

**ข้อดี:**
- สามารถแก้ไขเวลาเดิมพร้อมกับเพิ่มรายการใหม่
- มองเห็นข้อมูลเดือนทั้งหมดขณะเลือกวัน

**หมายเหตุ:** เมื่อไม่มีข้อมูลสำหรับวันที่คลิก modal จะยังแสดง 1 แถวว่างเพื่อให้กรอกข้อมูลใหม่

---

### วิธีที่ 3: แก้ไขเวลาที่มีอยู่

```
1. คลิก "แก้ไขเวลา"
   ↓
2. คลิกที่วันที่มีข้อมูล (จำนวนชั่วโมงจะแสดง)
   ↓
3. Modal เปิดขึ้นมากับข้อมูลเดิม
   ↓
4. แก้ไขข้อมูลที่ต้องการ
   ↓
5. คลิก "บันทึก"
```

## ข้อจำกัด และ ข้อมูลเพิ่มเติม (Constraints & Info)

| ข้อมูล | ข้อจำกัด | ข้อกำหนด |
|--------|--------|---------|
| **ชั่วโมง** | 0 - ไม่นับ | ต้อง > 0 |
| **วันที่** | เลือกได้ | ต้องในเดือนปัจจุบัน |
| **โครงการ** | จำเป็น | ต้องมีโครงการ 1 รายการขึ้นไป |
| **Task** | ทำเลือก | ถ้าไม่เลือก = General Work |
| **Start/End Time** | ทำเลือก | ถ้าเลือก ต้องมี Start ก่อน End |
| **สถานะ** | Auto | Draft (ตอนสร้าง) |

## Flow สมบูรณ์ (Complete Flow)

```
┌─────────────┐
│  เปิดหน้า   │
│  Timesheet  │
└──────┬──────┘
       │
       ├─────────────────────────┐
       │                         │
       ▼                         ▼
   [เพิ่มรายการ]            [แก้ไขเวลา]
   ใหม่ (Quick Add)         (Edit Mode)
       │                         │
       ├────────────┬────────────┤
       │            │            │
       ▼            ▼            ▼
    Modal      Modal        Modal
   (วันนี้)   (วันใหม่)    (ข้อมูลเดิม)
       │            │            │
       └────────────┼────────────┘
                    │
                    ▼
            [บันทึก + ปิด]
                    │
                    ▼
            ตารางอัปเดต
                    │
                    ▼
            [ส่งอนุมัติ]
```

## สถานะ (Status)

```
Draft (แบบร่าง)
    ↓ [ส่งอนุมัติ]
Submitted (รออนุมัติ)
    ├─ [อนุมัติ] ↓ Approved
    └─ [ปฏิเสธ] ↓ Rejected (กลับมา Draft)
```

สามารถแก้ไขได้: **Draft**, **Rejected**, **Submitted** (ถ้า admin/manager)

## ข้อเสนอแนะ (Tips)

✓ ใช้ "เพิ่มรายการใหม่" สำหรับการเพิ่มอย่างรวดเร็ว
✓ ใช้ "แก้ไขเวลา" เมื่อต้องแก้ไขหลายรายการในเดือน
✓ ส่งอนุมัติก่อนสิ้นเดือนเพื่อหลีกเลี่ยงอุปสรรค
✓ เลือก Task ให้ชัดเจนเพื่อการติดตามโครงการที่ดี
✓ เพิ่มคำอธิบายสำหรับงานที่เป็นพิเศษหรือสำคัญ
