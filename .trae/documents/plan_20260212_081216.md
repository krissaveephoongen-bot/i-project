# แผนการล้างระบบและทดสอบ End-to-End (Reset & Golden Path Test)

เพื่อให้ได้ผลลัพธ์ที่ถูกต้องและปลอดภัย ผมจะสร้าง **Automated Script** ผ่าน API พิเศษเพื่อจัดการล้างข้อมูลและสร้างข้อมูลตัวอย่างใหม่ทั้งหมดในชุดเดียว (Golden Path) ตามที่คุณต้องการ

## ขั้นตอนการดำเนินการ

### 1. สร้างระบบล้างและเตรียมข้อมูล (Reset & Seed API)
ผมจะสร้าง API พิเศษ (`/api/admin/reset-and-seed`) ที่ทำงานดังนี้:
*   **Clean Data:** ลบข้อมูลตามลำดับเพื่อไม่ให้ติด Error (Foreign Keys):
    1.  `timesheets` & `submissions` (เวลาทำงาน)
    2.  `tasks` & `logs` (งานและประวัติ)
    3.  `milestones` (งวดงาน)
    4.  `risks` & `documents` (ความเสี่ยงและเอกสาร)
    5.  `audit_logs` (ประวัติการใช้งาน)
    6.  `projects` (โครงการ)
    7.  `clients` (ลูกค้า)
    *   *หมายเหตุ: ข้อมูล Users จะถูกเก็บไว้ครบถ้วน*

*   **Seed Golden Data (สร้างข้อมูลชุดทดสอบ):**
    1.  สร้างลูกค้า: **"Golden Customer Co., Ltd."**
    2.  สร้างโครงการ: **"Project Phoenix (Rebirth)"** (งบ 1,000,000 บาท)
    3.  สร้าง Milestone: **"Phase 1: Requirements"** (30%)
    4.  สร้าง Task: **"Gather Requirements"** (Assign ให้ User คนแรกในระบบ)
    5.  ลงเวลา (Timesheet): ลงเวลา **8 ชั่วโมง** ใน Task นี้
    6.  จำลองการอนุมัติ (Approval): ปรับสถานะเป็น **Approved**

### 2. รันคำสั่งทดสอบ (Execution)
*   เรียกใช้งาน API นี้เพื่อทำการล้างและสร้างข้อมูลใหม่ทันที
*   ระบบจะคืนค่า **Report** กลับมาว่าสร้างข้อมูลอะไรไปบ้าง (พร้อม ID)

### 3. ตรวจสอบผลลัพธ์ (Verification)
หลังจากรันเสร็จ คุณสามารถเข้าไปตรวจสอบที่หน้าเว็บได้ทันที:
*   **Projects:** เห็นโครงการ "Project Phoenix"
*   **Tasks:** เห็นงาน "Gather Requirements"
*   **Timesheets:** เห็นเวลาที่ลงไว้ 8 ชม.
*   **Reports:** กราฟและตัวเลขต้องขึ้นถูกต้องตามข้อมูลชุดนี้

คุณยืนยันให้ผมเริ่มดำเนินการ **ล้างและสร้างข้อมูลใหม่** ตามแผนนี้เลยไหมครับ?
