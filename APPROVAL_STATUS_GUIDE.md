# Admin Approval Status Page - คู่มือการใช้งาน

## 📍 ตำแหน่ง (Location)

**Menu Path:** Admin Console → Approval Status
**URL:** `/admin/approvals`
**Access:** เฉพาะ Admin เท่านั้น

---

## 🎯 วัตถุประสงค์ (Purpose)

หน้าสรุปสถานะคำร้องการอนุมัติทั้งหมด เพื่อให้ Admin สามารถ:
- ✅ ตรวจสอบสถานะคำร้องการอนุมัติทั้งหมด (ไทม์ชีท, ค่าใช้จ่าย, ลากิจ)
- ✅ จัดการคำร้องที่รอการอนุมัติ
- ✅ ติดตามสถิติการอนุมัติทั้งหมด
- ✅ ค้นหาและกรองข้อมูลอย่างรวดเร็ว

---

## 📊 องค์ประกอบหลัก (Main Components)

### 1. **Statistics Cards (บัตรสถิติ)**

```
┌──────────────┐  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐
│  รวมทั้งหมด  │  │ รอการอนุมัติ  │  │  อนุมัติแล้ว  │  │ ปฏิเสธแล้ว   │
│    150       │  │     45       │  │     85       │  │      20      │
└──────────────┘  └──────────────┘  └──────────────┘  └──────────────┘
   ทั้งหมด         เหลือทำ           สำเร็จ              ไม่อนุมัติ
```

**ตัวชี้วัด:**
- **รวมทั้งหมด:** จำนวนคำร้องทั้งหมด
- **รอการอนุมัติ:** คำร้องที่ยังรอการอนุมัติ (Pending)
- **อนุมัติแล้ว:** คำร้องที่ได้รับการอนุมัติ
- **ปฏิเสธแล้ว:** คำร้องที่ปฏิเสธ

---

### 2. **Filter Controls (ตัวควบคุมการค้นหา)**

```
┌─────────────────────┐  ┌──────────────────┐  ┌──────────────────┐
│ ค้นหาชื่อผู้ใช้/ID  │  │ ประเภท: ทั้งหมด  │  │ สถานะ: ทั้งหมด   │
│  [John | Task123]   │  │ [▼ ไทม์ชีท]      │  │ [▼ รอการอนุมัติ]  │
│                     │  │   ค่าใช้จ่าย      │  │   อนุมัติแล้ว      │
│                     │  │   ลากิจ           │  │   ปฏิเสธแล้ว      │
└─────────────────────┘  └──────────────────┘  └──────────────────┘
```

**ฟังก์ชัน:**

#### Search Box (กล่องค้นหา)
- ค้นหาจาก: ชื่อผู้ใช้ หรือ ID ของคำร้อง
- Live search (ค้นหาขณะพิมพ์)

#### Type Filter (กรองประเภท)
```
ทั้งหมด (All)
├─ ไทม์ชีท (Timesheet) - สีน้ำเงิน
├─ ค่าใช้จ่าย (Expense) - สีเขียว
└─ ลากิจ (Leave) - สีม่วง
```

#### Status Filter (กรองสถานะ)
```
ทั้งหมด (All)
├─ รอการอนุมัติ (Pending) - สีเหลือง
├─ อนุมัติแล้ว (Approved) - สีเขียว
└─ ปฏิเสธแล้ว (Rejected) - สีแดง
```

---

### 3. **Approval Table (ตารางคำร้อง)**

#### ส่วนหัวตาราง (Table Headers)
| ประเภท | ผู้ใช้ | วันที่ | จำนวน | สถานะ | ส่งเมื่อ | การกระทำ |
|-------|------|-------|------|-------|--------|---------|
| Type  | User | Date  | Amount/Hours | Status | Timestamp | Action |

#### ตัวอย่างแถว (Example Row)
```
┌─────────┬───────────┬────────────┬────────┬─────────┬──────────────┬────────┐
│ ไทม์ชีท │ สมหมาย    │ 15 ก.พ. 24 │ 8 ชม.  │ รออนุมัติ │ 14 ก.พ. 10:30 │ ดู → │
│ ค่าใช้จ่าย│ นาย โรจน์ │ 16 ก.พ. 24 │ ฿5,000 │ อนุมัติแล้ว │ 15 ก.พ. 14:15 │ ดู → │
│ ลากิจ   │ นางสาว ก │ 17 ก.พ. 24 │   -    │ ปฏิเสธแล้ว │ 16 ก.พ. 09:00 │ ดู → │
└─────────┴───────────┴────────────┴────────┴─────────┴──────────────┴────────┘
```

**ข้อมูลที่แสดง:**
- ประเภท: ไทม์ชีท/ค่าใช้จ่าย/ลากิจ
- ผู้ใช้: ชื่อผู้ส่งคำร้อง
- วันที่: วันที่เกี่ยวข้องกับคำร้อง
- จำนวน: 
  - สำหรับค่าใช้จ่าย: จำนวนเงิน (฿)
  - สำหรับไทม์ชีท: ชั่วโมง (ชม.)
  - สำหรับลากิจ: -
- สถานะ: 
  - 🟡 รอการอนุมัติ (Pending)
  - 🟢 อนุมัติแล้ว (Approved)
  - 🔴 ปฏิเสธแล้ว (Rejected)
- ส่งเมื่อ: วันที่และเวลาที่ส่งคำร้อง
- การกระทำ: ปุ่มดู (ไปยังหน้ารายละเอียด)

## 🎮 วิธีการใช้งาน (Usage Instructions)

### ขั้นตอนที่ 1: เข้าสู่หน้า Approval Status
```
1. คลิกเมนู Admin
2. เลือก "Approval Status"
3. หน้าจะโหลดและแสดงข้อมูลทั้งหมด
```

### ขั้นตอนที่ 2: ตรวจสอบสถิติ
```
ดูการ์ดสถิติ 4 ใบ:
- รวม: 150 รายการ
- รอ: 45 รายการ ⚠️ (ต้องดำเนินการ)
- อนุมัติแล้ว: 85 รายการ ✓
- ปฏิเสธแล้ว: 20 รายการ ✗
```

### ขั้นตอนที่ 3: ค้นหาคำร้อง (ตัวเลือก)
```
วิธีที่ 1: ค้นหาชื่อผู้ใช้
- พิมพ์ชื่อในกล่องค้นหา
- เช่น: "สมหมาย"

วิธีที่ 2: ค้นหา ID
- พิมพ์ ID ในกล่องค้นหา
- เช่น: "TS-12345"
```

### ขั้นตอนที่ 4: กรองตามประเภท (ตัวเลือก)
```
คลิกเมนู "ประเภท":
- ทั้งหมด (default)
- ไทม์ชีท
- ค่าใช้จ่าย
- ลากิจ
```

### ขั้นตอนที่ 5: กรองตามสถานะ (ตัวเลือก)
```
คลิกเมนู "สถานะ":
- ทั้งหมด (default)
- รอการอนุมัติ (focus ที่นี่)
- อนุมัติแล้ว
- ปฏิเสธแล้ว
```

### ขั้นตอนที่ 6: ดูรายละเอียด
```
1. คลิกปุ่ม "ดู" ที่คำร้องที่ต้องการ
2. ไปยังหน้ารายละเอียดการอนุมัติ
3. สามารถอนุมัติ/ปฏิเสธได้จากที่นั่น
```

## 🔍 ตัวอย่างการใช้ (Use Case Examples)

### Scenario 1: ตรวจสอบคำร้องที่รอการอนุมัติ
```
1. เปิด Approval Status
2. ดูการ์ด "รอการอนุมัติ" → 45 รายการ
3. เลือก Filter "สถานะ: รอการอนุมัติ"
4. ตารางจะแสดงเฉพาะคำร้องที่รอ
5. คลิก "ดู" ที่คำร้องแต่ละรายการ
6. ทำการอนุมัติหรือปฏิเสธ
```

### Scenario 2: ค้นหาคำร้องของเฉพาะคน
```
1. พิมพ์ชื่อผู้ใช้ในกล่องค้นหา เช่น "สมหมาย"
2. ตารางจะกรองและแสดงเฉพาะของคนนั้น
3. ตรวจสอบประวัติการอนุมัติของเขา
4. คลิก "ดู" ถ้าต้องการเข้าไปดูรายละเอียด
```

### Scenario 3: ตรวจสอบค่าใช้จ่ายที่รอ
```
1. เลือก Filter "ประเภท: ค่าใช้จ่าย"
2. เลือก Filter "สถานะ: รอการอนุมัติ"
3. ตารางแสดงเฉพาะค่าใช้จ่ายที่รอ
4. สามารถเรียงตามจำนวนเงินได้
5. ทำการอนุมัติหรือปฏิเสธตามสมควร
```

### Scenario 4: ดูสรุปสถิติรายสัปดาห์
```
1. เปิด Approval Status
2. ตรวจสอบการ์ดสถิติ:
   - ขาดหลัง 45 รายการ?
   - กำลังทำได้ดีขึ้นหรือเลา?
3. Compare กับสัปดาห์ที่แล้ว
4. Plan ขนาดทีมหรือแนะนำการปรับปรุง
```

## ⚙️ เทคนิคและ Tips

### Live Refresh
- หน้าจะ refresh โดยอัตโนมัติทุก 5 วินาที
- ไม่ต้องรีเฟรชเอง มันจะอัปเดตตัวเอง

### Color Coding (รหัสสี)
```
ประเภท:
🔵 ไทม์ชีท (Timesheet) - Blue
🟢 ค่าใช้จ่าย (Expense) - Green
🟣 ลากิจ (Leave) - Purple

สถานะ:
🟡 รอการอนุมัติ (Pending) - Yellow
🟢 อนุมัติแล้ว (Approved) - Green
🔴 ปฏิเสธแล้ว (Rejected) - Red
```

### Keyboard Shortcuts (ตัวเลือก)
```
Ctrl+F: เปิดค้นหาในเบราว์เซอร์
(ดีสำหรับค้นหาใน Table)
```

### Export Data (ตัวเลือก)
- ตารางสามารถ Copy ได้ (Ctrl+C)
- นำไป Excel ได้ถ้าต้องการ

## 🐛 Troubleshooting

### ปัญหา: ไม่เห็นข้อมูล
```
✓ ตรวจสอบว่า login แล้ว
✓ ตรวจสอบว่าเป็น Admin role
✓ ลองรีเฟรชหน้า (Ctrl+F5)
✓ ตรวจสอบ Network tab ว่า API ตอบ
```

```
✓ กด Enter หลังค้นหา
✓ ลองเปลี่ยน Filter อื่น
✓ ลองเลือก "ทั้งหมด" ใหม่
```

### ปัญหา: ปุ่ม "ดู" ไม่ทำงาน
```
✓ ตรวจสอบ Internet connection
✓ ลองคลิกใหม่
✓ ปิด popup blocker
```
