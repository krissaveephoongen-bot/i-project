API ENDPOINTS (Quick Reference)
================================================================================

All endpoints use JSON and are located at: /api/prisma/

LISTING COSTS:
  GET /costs?status=pending                    → Get pending costs
  GET /costs?category=Software                 → Filter by category
  GET /costs?startDate=2024-01-01&endDate=...  → Date range filter

MANAGING COSTS:
  POST /costs                                  → Create new cost
  PUT /costs/:id                               → Update cost
  DELETE /costs/:id                            → Delete cost

APPROVALS:
  POST /costs/:id/approve                      → Approve or reject
  GET /pending-approvals                       → Get pending list
  GET /approval-statistics                     → Stats

SUMMARIES:
  GET /costs/project/:projectId/summary        → Project summary

See: API_TESTING_GUIDE.md for complete documentation
